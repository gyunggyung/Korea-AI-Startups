## Introduction

- 대한민국에 존재하는 **다양한 스타트업**들을 기록합니다.

- 해당 기록을 통해 취업 준비를 하시는 분들, 이직을 준비하시는 분들에게 선택의 폭이 늘어나기를 바랍니다.

- 기업에 대한 질문을 [**issues**](https://github.com/gyunggyung/Korea-AI-Startups/issues)에 올려주시면, 최대한 답변 드리겠습니다.
